# cool-vue-template
vue2.x template with webpack
## Usage
This is a project template for vue-cli.
```
npm install -g vue-cli
vue init webpack-simple my-project
cd my-project
npm install
npm run dev
```
## start
```
npm run build:dll
npm run build
npm run dev
```
## rc
```
npm run build:rc
```
## online
```
npm run build:prod
```
